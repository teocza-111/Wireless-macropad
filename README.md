# Wireless-macropad
It's a basic 3x3 keyboard with bluetooth connectivity and a rotary encoder built on ZMK firmware.
## It features a(n):
- nrf52840 microcontroller
- Seeed 106990290 lipo charger
- EC11E Rotary Encoder w/ switch
- Li-Po Battery 4000mah
### Why?
I like simracing but i needed some extra keybinds so i made my first vesion of this project but it was annoying that there was an extra cable on my desk so now im making it wireless.
