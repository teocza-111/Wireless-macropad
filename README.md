# Wireless-macropad
It's a basic 3x3 keyboard with bluetooth connectivity and a rotary encoder.
## It features a:
- nrf52840 microcontroller
- Seeed 106990290 lipo charger
- EC11E Rotary Encoder w/ switch
- Li-Po Battery 1000mah
